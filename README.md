# Setup
Before your interview make sure to clone this repo locally and get this basic app running.

* Make sure to have `npm` installed:
```
$ brew update
$ brew install npm
```
* Clone this repo
* Run `npm start`
* Make sure [http://localhost:3000](http://localhost:3000) loads in the browser.
